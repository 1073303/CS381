# CS381

開放平台軟體final-project:Line Bot 自動對發票機器人

### 簡介
透過Python網路爬蟲財政部公開資料，結合Line messaging API 實做一個簡易的發票兌獎機器人

### 使用說明
1.掃描QRcode將Line bot 機器人加入好友

2.相關指令

├── @輸入發票最後三碼           
├── @前期中獎號碼              
├── @本期中獎號碼

在輸入@前期中獎號碼/@本期中獎號碼後，過一段時間會顯示出相對應的資訊，@輸入發票最後三碼則會顯示出輸入末三碼的訊息，依指令輸入後即可判斷是否有中獎
